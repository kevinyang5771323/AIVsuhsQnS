# 生鲜超市管理系统 java700

## 项目概述

生鲜超市管理系统是一款基于Java和SpringBoot框架，结合VUE实现前后端分离的B/S架构系统。该系统致力于为生鲜超市提供全面的管理解决方案，涵盖用户管理、商品管理、采购与库存管理等多个方面。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**: IDEA、JDK1.8、Maven、MySQL5.7+
- **技术框架**: Java、MySQL、SSM、Mybatis-Plus、VUE、jQuery、HTML
- **数据库可视化工具**: Navicat
- **服务器**: SpringBoot内嵌Apache Tomcat

## 功能模块

系统角色包括管理员、用户和前台。

### 一、通用功能

- 登录功能
- 注册功能

### 二、管理端

- 首页
- 个人中心
  - 用户管理
  - 生鲜仓库管理
  - 采购单登记管理
  - 入库信息管理
  - 出库信息管理
  - 产品分类管理
  - 生鲜产品管理
  - 系统管理
  - 订单管理

### 三、用户端

- 系统首页
- 个人中心

### 四、前台功能

- 首页
- 个人中心
- 生鲜产品
- 新闻公告
- 购物车

系统支持在Windows和Mac操作系统环境下运行。
